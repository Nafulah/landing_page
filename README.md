## 📌 Project Description

Sifa Online Store is a mobile-first, responsive e-commerce landing page.  
The goal: Showcase clean UI, responsive design, and a pink brand identity for a Kenyan online shop.

This is a portfolio project for my Full Stack Web Developer certificate + BIT degree.

## ✨ Key Features

- *Responsive Navbar*: Hamburger menu ☰ that drops down on screens < 768px
- *Products Section*: 3-column product card grid on desktop, stacks on mobile
- *About Us Section*: Styled with brand accent and readable typography
- *Contact Section*: UI for a contact form, ready for backend connection
- *Footer Section*: Copyright, Quick Links, Social Icons, and Brand info
- *Mobile-First CSS*: Tested on Desktop, Tablet 768px, and Mobile 480px

## 🚀 How to Run Locally

1. Clone the repo: git clone <https://github.com/Nafulah/sifa-online-store.git>
2. Open index.html in your browser.
3. Or use VS Code: Install Live Server extension > Go Live

To test responsiveness: Right-click > Inspect > Toggle Device Toolbar
